# 🧩 practice_codingtest

파이썬으로 코딩테스트를 연습하는 저장소입니다.  
**「Do it! 알고리즘 코딩테스트 - 파이썬 편」**을 참고해 문제를 풀고 정리합니다.

> 이 레포의 코드는 책과 **완전히 동일할 때도 있고**,  
> 이해/최적화/개인 스타일에 따라 **다르게 작성된 버전도 포함**됩니다.

## 📌 구성(예시)
- 문제 풀이 코드 (Python)
- 풀이 과정 메모 / 주석
- 필요하면 같은 문제의 여러 풀이(기본/개선/다른 접근)

## ✅ 목표
- 기본 알고리즘/자료구조 구현에 익숙해지기
- 문제 풀이 패턴 체득
- 시간/메모리 관점에서 개선하는 습관 만들기
